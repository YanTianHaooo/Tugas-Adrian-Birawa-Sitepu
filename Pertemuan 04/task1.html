Belajar Konsep Dasar CSS

CSS (Cascading Style Sheets) digunakan untuk mengatur tampilan dan estetika sebuah halaman web. Dengan CSS, kita bisa memisahkan konten (HTML) dari desain presentasionalnya.

Memahami Selector ID dan Class

Selector Class diawali dengan tanda titik (.) dan bisa digunakan berulang kali pada banyak elemen. Sementara Selector ID diawali dengan tanda pagar (#) dan bersifat unik, hanya boleh digunakan sekali per halaman web.

Pentingnya Semantic HTML

Tag semantik seperti header, nav, main, dan article membantu mesin pencari (SEO) serta perangkat pembaca layar (screen reader) memahami struktur dan konteks halaman web kita dengan lebih baik.
